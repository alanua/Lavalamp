# Lavalamp

Firmware and visual-effects project for a custom cylindrical 16x16 LED lamp based on WLED.

Physical target: matte cylindrical glass with internal diffuser film from an LED TV screen/backlight stack. The diffuser blurs pixels so the lamp shows soft lava, bloom, fog and liquid-light effects instead of a sharp pixel matrix.

Hardware target: ESP32 first. ESP8266 support only where practical.
